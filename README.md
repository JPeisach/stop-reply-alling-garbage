# [stopreplyallinggarbage.com](https://stopreplyallinggarbage.com)
An informational website about why you should stop reply-alling in emails,
even when sending jokes. Forked and inspired by dontasktoask.com.

If you want to translate, go ahead, hopefully maybe hosting on a good domain
can be found.
